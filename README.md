# Chart2PNG

Chart2PNGは某音楽ゲームシミュレーターで使用される譜面ファイル（`.mcz`）を読み込み、Keyモードの譜面画像として表示・保存するツールです。

## 概要

MC形式の譜面データを解析し、ブラウザ上でSVGとして描画します。
簡単に言えば[某譜面保管所](https://sdvx.in/)にあるような見た目の画像を作ります。
描画した譜面はPNG画像として保存できます。

## 主な機能

* `.mcz` ファイルの読み込み
* 譜面のSVG描画
* PNG画像として保存

## 対応形式

入力:

* `.mcz`

出力:

* `.png`

## 使い方
予めmczファイルは作成しておいてください
1. アプリを起動
2. `.mcz` ファイルを選択
3. 譜面が表示されることを確認
4. 「PNG保存」を押して画像を保存

## 保存ファイル名

保存される画像ファイル名は以下の形式になります。

```text
曲名_難易度名_UNIX時間.png
```

例:

```text
DREAM_KEY-BEGINNER-Lv.2_3553550000.png
```


空白は `-` に変換され、一部のOSで使用できない文字は自動的に除去されます。
